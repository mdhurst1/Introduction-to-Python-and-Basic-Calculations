# Introduction-to-Python-and-Basic-Calculations
This interactive python notebook serves as an Introduction to Python for geoscientists.
It is an exercise in EARTH4090 Landscape Dynamics, an honours level course on the Environemental Geoscience degree in the School of Geographical and Earth Sciences at the Univrsity of Glasgow.

## Overview of the lab
In this lab we will start to explore some stuff about Tectonic Geomorphology. In particular it is important to get used to calculating distances, rates, forces, and other relevant things that we can quantify from remote sensing data, topographic data or other sources of landscape information.

The model is written in the Python computer programming language, but you don't need any prior knowledge of computer programming. 

### Intended Learning Outcomes
The intended learning outcomes of this lab are:
* To understand how to conduct basic calculations in Python
* To be able to work out units of measurement for calculations of forces and pressures
* To have knowledge of how scientific numbers are represented in scientific computing
